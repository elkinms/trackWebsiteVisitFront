React + Vite frontend for tracking website visits by country.

## Features
- React SPA built with Vite
- Fetches statistics from backend API
- Environment-based configuration (.env)

## Installation

```bash
# Clone repository
git clone https://github.com/elkinms/trackWebsiteVisitFront
cd track-website-visit-front
# Install dependencies
npm install
# Running in development mode
npm run dev
# Build for production
npm run build
# Preview the production build
npm run preview
```
